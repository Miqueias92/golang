# golang

## installation guide

https://go.dev/doc/install

## run 
` 
 go run hello-world/hello.go
`

### build
` 
 go build hello-word/hello.go
` 

### run builded file
`
./hello
`

### create a module

`
  go mod init [module_name]     
`

### build module
`
  go build     
`

### run
`
 ./[module_name]   
`

## install packages

`
go get github.com/google/uuid
`
